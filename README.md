# 給料カレンダー

複数のバイト先を登録し、カレンダーに勤務時間を入力して月ごとの労働時間と給料を確認できるWebサイトです。

## 主な機能

- 複数バイト先の登録
- 基本時給・深夜時給・交通費の設定
- 深夜時給を適用する開始・終了時刻をバイト先ごとに設定
- 20色のカラーパレットから表示色を選択
- よく使うバイト先・勤務時間・休憩をプリセット登録してクイック入力
- 22時以降と翌5時までの深夜勤務を自動計算
- 予定勤務と実績勤務の区別
- 月別・バイト別の労働時間と給料集計
- 月収目標と進捗
- JSONバックアップ・復元
- CSV出力
- スマホ対応
- ブラウザ内保存（localStorage）

## GitHub Pagesで公開する方法

1. GitHubで新しいリポジトリを作る
2. `index.html`、`style.css`、`script.js`をアップロードする
3. リポジトリの `Settings` を開く
4. 左側の `Pages` を押す
5. `Build and deployment` の `Source` を `Deploy from a branch` にする
6. Branchを `main`、フォルダを `/(root)` にして保存する
7. 数分後に表示されたURLを開く

## 注意

入力内容は使用中のブラウザに保存されます。ブラウザのデータを削除すると消えることがあるため、定期的に「バックアップ」ボタンで保存してください。


## v3追加機能
- 勤務カレンダーと給料日カレンダーの表示切替
- バイト先ごとの締め日・給料日・当月/翌月支給設定
- プリセットを押すだけで勤務を即時登録
- 予定/実績区分を廃止


## プリセット入力

勤務カレンダーで日付を選び、表示されたプリセットを押すと、その日付に勤務が即登録されます。
保存ボタンは不要です。手入力するときだけ保存ボタンを使います。
クイック入力欄はありません。
